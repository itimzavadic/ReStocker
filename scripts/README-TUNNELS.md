# Инструкция по запуску туннелей

## Автоматический запуск (рекомендуется)

Запустите скрипт:
```powershell
.\scripts\start-tunnels.ps1
```

Скрипт откроет два окна PowerShell с туннелями.

## Ручной запуск

### Backend туннель (порт 8000)
```powershell
ssh -R 80:localhost:8000 nokey@localhost.run
```

### Frontend туннель (порт 3000)
```powershell
ssh -R 80:localhost:3000 nokey@localhost.run
```

## Почему туннели закрываются?

- Нестабильность localhost.run (бесплатный сервис)
- Неактивность (туннели могут закрываться при отсутствии трафика)
- Сетевые проблемы

## Решение: Деплой на стабильный хостинг

Для продакшена рекомендуется использовать:
- Railway.app (бесплатный план)
- Render.com (бесплатный план)
- Fly.io (бесплатный план)
